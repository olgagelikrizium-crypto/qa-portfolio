Intensive "Web Testing Program"

This intensive QA course focused on practical manual testing skills, real-world scenarios, and industry tools. During the course, I worked with web application, analyzed functionality, identified defects, and practiced structured bug reporting following professional standards.

3 weeks
  • Performed manual testing of web platforms, focusing on core user flows and business logic
  • Identified, documented, and reported bugs using structured bug report formats
  • Practiced writing clear and detailed bug reports with steps to reproduce, expected and actual results
  • Worked with different testing techniques including exploratory testing and functional testing
  • Analyzed edge cases and user behavior scenarios to uncover hidden issues
  • Used browser DevTools to inspect elements, analyze network requests, and debug UI issues
  • Learned basics of SQL and wrote simple queries to retrieve and analyze data
  • Tested UI components for usability, layout issues, and responsiveness
  • Practiced working with test scenarios and test cases
  • Gained experience in analyzing requirements and verifying expected system behavior
